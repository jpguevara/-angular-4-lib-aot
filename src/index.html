Angular Library Test Harness

This is a runing sample of the library.

Here are some quick facts about it:

  • This is an angular library project.
  • The goal of this project is to be consumed by another angular application.
  • This page will not be included into the final bundle.
  • This project compiles into a set of *.js and *.d.ts files.

To execute the unti test for the project run this command. npm run test

The files to distribute and publish are located in: dist/ after the build command.

The Entry component for the test harness is the AngularLibComponent
